# Council Agenda

Strategic topics queued for future council sessions. Unlike reactive discussions, agenda items allow for proactive planning and deliberation.

## Format
```markdown
## [Priority] Topic Title
- **Added**: YYYY-MM-DD
- **Context**: Why this needs council discussion
- **Suggested Stance**: balanced | critical | adversarial
- **Related Decisions**: Links to relevant past decisions
```

## Priority Levels
- **P0**: Blocking issues requiring immediate council attention
- **P1**: Important topics that should be addressed soon
- **P2**: Nice-to-have discussions when time permits
- **Parked**: Ideas to revisit later (not urgent)
